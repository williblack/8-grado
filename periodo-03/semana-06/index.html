¿Por qué diseñar formatos?

En la clase anterior usamos Filtros en Excel para buscar datos en una tabla gigante (BD). Pero el resultado es solo una fila aburrida.

Concepto Clave: Un formato transforma una Base de Datos en una Información Útil. Es pasar de una tabla llena de celdas a un carnet, una factura o una ficha médica visual.
🤖 Trabajo en sala (IA)

La IA como Arquitecto de Datos

Para crear un formato profesional, primero debemos saber qué campos incluir. Usa este "Prompt" con tu IA:

"Actúa como experto en Excel. Necesito crear un formato de consulta para una [FICHA MÉDICA]. Genera una tabla con 5 datos personales, 3 familiares y 2 académicos que no pueden faltar."
