📓 Resumen de Conexión

En el periodo pasado aprendimos que una Base de Datos es un sistema para organizar información. Ahora, usaremos Excel para darle vida a esos datos.

  • 1. Campo (Columna): Define el tipo de dato (Nombre, Edad, Precio). En Excel se identifican con letras (A, B, C).
  • 2. Registro (Fila): Es el conjunto de datos de un objeto o persona. En Excel se identifican con números (1, 2, 3).
  • 3. Dato (Celda): El valor específico en la intersección de una fila y columna.
