Resumen de Conceptos Clave

🔄 ¿Qué aprendimos el periodo pasado?

Una Base de Datos es un sistema organizado para guardar información. Para que sea eficiente, debe tener una estructura clara. No es solo "guardar", es saber "dónde" está cada cosa.

  • ✔️ Tabla: El contenedor del tema.
  • ✔️ Campo: El título de la columna (Atributo).
  • ✔️ Registro: La fila completa con los datos de alguien.

🚀 Rumbo a Microsoft Excel
